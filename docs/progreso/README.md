# Progreso del proyecto - Cadena de Tierra

Esta carpeta se usa **solo para documentar el avance real del proyecto**.

## Estructura

- `00-roadmap.md` → etapas, pasos y criterio para marcar avance
- `01-backlog.md` → pendientes inmediatos
- `02-log.md` → bitácora cronológica de avances reales
- `03-checklist-demo.md` → checklist para validar demo funcional
- `04-links.md` → links importantes del proyecto

## Regla de uso

Cada vez que completemos algo importante:
1. Se actualiza la bitácora en `02-log.md`
2. Se marca el checklist correspondiente
3. Se actualiza el backlog si cambió la prioridad
